# 📊 AdTech CTR Analysis Project

This is a simple Python project to calculate Click Through Rate (CTR) for ads.

## 🚀 Features
- Calculate CTR (Clicks / Impressions)
- Find best performing ad
- Rank ads based on CTR
- Visualize CTR using bar chart

## 🛠 Tech Used
- Python
- Pandas
- Matplotlib

## 📌 Output
- CTR calculation
- Best ad selection
- Graph visualization

## ▶️ How to Run
1. Install dependencies:
   pip install pandas matplotlib

2. Run:
   python ctr_analysis.py
